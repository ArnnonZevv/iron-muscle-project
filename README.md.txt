# Iron Muscle Gym Membership System  
A full-stack web application built using **React (Vite)** for the front-end and **Spring Boot** for the back-end.  
This project is based on the system requirements defined in the Iron Muscle PDF proposal, focusing on gym membership management, class scheduling, trainer appointments, payments, and cancellation processes.
